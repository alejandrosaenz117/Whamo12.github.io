# [Professional Portfolio](https://www.alejandrosaenz.com/)

My personal portfolio and resume website showcasing experience in application security, software engineering, and open source contributions.

## Features

- Responsive design (mobile-first)
- Professional experience and job history
- Open source projects showcase with GitHub stats
- Photography gallery with carousel
- Speaking engagements and content links
- Skill tags with icons

## Tech Stack

- HTML5, CSS3, JavaScript
- Bootstrap 5
- GitHub Pages (GitHub Actions deployment)
- Custom domain with HTTPS

## Build & Deploy

The site automatically deploys to GitHub Pages on every push to `main`. No manual build step required.
